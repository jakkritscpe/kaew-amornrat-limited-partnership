# หจก.แก้วอมรรัตน์ - เว็บไซต์บริการไอที

เว็บไซต์แนะนำบริการของ **ห้างหุ้นส่วนจำกัด แก้วอมรรัตน์** ให้บริการไอทีครบวงจรแบบ Onsite ทั่วกรุงเทพและปริมณฑล

## เกี่ยวกับโปรเจกต์

เว็บไซต์ Single Page Application (SPA) สำหรับแสดงข้อมูลบริการไอทีของบริษัท ออกแบบมา Responsive รองรับทั้งมือถือและคอมพิวเตอร์ มีอนิเมชันเลื่อนหน้าจอสวยงาม

## บริการที่นำเสนอบนเว็บ

- **Computer** - ซ่อมคอมพิวเตอร์ โน๊ตบุ๊ค เปลี่ยนอะไหล่ ลงโปรแกรม กำจัดไวรัส
- **CCTV** - ติดตั้งกล้องวงจรปิด ระบบ DVR/NVR ดูผ่านมือถือ
- **Server** - ติดตั้งและดูแลเซิร์ฟเวอร์ กำหนดสิทธิ์ผู้ใช้ มอนิเตอริ่ง
- **Network LAN WiFi** - วางระบบสาย LAN ติดตั้ง WiFi ปรับแต่งสัญญาณ
- **Firewall** - ระบบความปลอดภัยเครือข่าย ป้องกันการโจมตี
- **NAS** - ระบบจัดเก็บข้อมูลแบบรวมศูนย์ แชร์ไฟล์ เข้าถึงระยะไกล
- **Backup Data** - ระบบสำรองข้อมูล กู้คืนข้อมูล คลาวด์แบ็คอัพ

## โครงสร้างหน้าเว็บ

| ส่วน | ไฟล์ | รายละเอียด |
|------|------|------------|
| แถบเมนู | `src/sections/Navigation.tsx` | เมนูด้านบน เปลี่ยนสีตามการเลื่อน รองรับเมนูมือถือ |
| หน้าแรก | `src/sections/Hero.tsx` | แบนเนอร์หลัก พร้อมปุ่มติดต่อ |
| บริการ | `src/sections/Services.tsx` | แสดงบริการทั้ง 7 ประเภท |
| ทำไมเลือกเรา | `src/sections/WhyChooseUs.tsx` | จุดเด่น 6 ข้อ เช่น ประสบการณ์ 10+ ปี, บริการ 24/7 |
| สถิติ | `src/sections/Statistics.tsx` | ตัวเลขผลงาน เช่น ลูกค้า 500+, ประสบการณ์ 10+ ปี |

| ท้ายเว็บ | `src/sections/Footer.tsx` | ลิงก์เมนู รายการบริการ ช่องทางโซเชียล |

## เทคโนโลยีที่ใช้

- **React 19** - ไลบรารีสร้าง UI
- **TypeScript** - ภาษา JavaScript แบบมี Type
- **Vite 7** - เครื่องมือ Build และ Dev Server ที่เร็ว
- **TailwindCSS 3** - เขียน CSS แบบ Utility-first
- **shadcn/ui** - คอมโพเนนต์ UI สำเร็จรูป (Button, Input, Select, Dialog ฯลฯ)
- **Lucide React** - ไอคอน
- **Recharts** - กราฟ/แผนภูมิ

## โครงสร้างโฟลเดอร์

```
src/
├── sections/          # ส่วนต่าง ๆ ของหน้าเว็บ (7 ไฟล์)
├── components/ui/     # คอมโพเนนต์ UI จาก shadcn/ui
├── hooks/             # Custom Hooks
├── lib/               # ฟังก์ชันช่วยเหลือ (utils)
├── App.tsx            # คอมโพเนนต์หลัก ประกอบทุก Section เข้าด้วยกัน
├── main.tsx           # จุดเริ่มต้นแอป
└── index.css          # สไตล์หลัก + TailwindCSS
```

## วิธีติดตั้งและรัน

### ติดตั้ง Dependencies

```bash
npm install
```

### รันโหมดพัฒนา (Development)

```bash
npm run dev
```

เปิดเบราว์เซอร์ไปที่ `http://localhost:5173`

### Build สำหรับใช้งานจริง (Production)

```bash
npm run build
```

ไฟล์ที่ Build เสร็จจะอยู่ในโฟลเดอร์ `dist/`

### ดูตัวอย่างหลัง Build

```bash
npm run preview
```

## คำสั่งทั้งหมด

| คำสั่ง | รายละเอียด |
|--------|------------|
| `npm run dev` | รัน Dev Server |
| `npm run build` | Build โปรเจกต์ |
| `npm run preview` | ดูตัวอย่างหลัง Build |
| `npm run lint` | ตรวจสอบโค้ดด้วย ESLint |
